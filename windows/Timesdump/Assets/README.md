# Assets

This directory should contain the following files for the Windows application:

## app.ico

A Windows icon file for the application. The icon should:
- Be in `.ico` format
- Contain multiple sizes (16x16, 32x32, 48x48, 256x256)
- Use a clock/timestamp-related design
- Follow Windows design guidelines

You can create this icon using tools like:
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [IcoFX](https://icofx.ro/)
- [Greenfish Icon Editor Pro](http://greenfishsoftware.org/gfie.php)

For development, you can use a placeholder icon or generate one from an SVG.
